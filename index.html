Ani-Search

Quickly search information on AniList

What is it?

Ani-Search is an unofficial browser extension for AniList which allows you to quickly search information on AniList.

The extension is available on Firefox and Chrome.

What's included?

Integrated search

Ani-Search comes with a light search interface which allows you to quickly search something on AniList and see results without having to go to the website itself.

You can search for anime, manga, studios, characters and staff. If you need to be a bit more specific with your search, you can also filter results by year and season for anime and manga.

If your are logged into AniList through Ani-Search, the extension will also return anime and manga that are in your lists.

Browser integration

Ani-Search adds a contextual menu that will appear when you selected a text on a page.

This contextual menu let you immediately search the selected text on AniList for all type of data available.

Browser search provider (FireFox only)

Ani-Search adds an anime search engine to the browser.

Starting your search using the keyword ani, the browser will automatically switch to this search engine and execute an anime search on AniList!
